# 🚀 选课社区快速启动

## 一键启动

```bash
# 1. 给脚本添加执行权限（仅需执行一次）
chmod +x start_jcourse.sh stop_jcourse.sh

# 2. 启动服务
bash start_jcourse.sh

# 3. 停止服务
bash stop_jcourse.sh
```

## 访问地址

启动成功后访问：

- **前端**: http://localhost:3000
- **后端API**: http://localhost:8000  
- **管理后台**: http://localhost:8000/admin
  - 用户名: `admin`
  - 密码: `admin`

## 系统要求

- macOS 系统
- 已安装 Homebrew

脚本会自动安装和配置所需的依赖。

详细说明请查看 `部署说明.md`
